使用git branch 可以查看分支
git branch develop可以创建分支
git checkout develop 可以切换分支
git branch -d 删除合并的分支
git branch -D 删除为何并的分支
git stash 可以暂时保存更改
git stash pop 可以恢复
git log 查看提交信息
git status 查看文件状态
git add 提交到暂存区
git commit -m (text)  提交到仓库
git rm --cached (html) 从暂存区删除
git checkout (html)  从暂存区取出,对工作区的文件进行覆盖
git reset --hard 可以恢复提交节点
